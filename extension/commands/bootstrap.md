# Bootstrap

Set up this repository to use the Spec Kit Memory workflow.

Tasks:
1. Ensure these folders exist:
   - docs/memory
   - specs
   - .github
2. Create missing memory files from the extension templates.
3. Create missing spec starter files from the extension templates.
4. Create or update `.github/copilot-instructions.md` to include the memory-aware workflow.
5. Summarize what was created and what should be customized first.

Prioritize preserving existing project files.
Never overwrite project-specific memory without explicit approval.
